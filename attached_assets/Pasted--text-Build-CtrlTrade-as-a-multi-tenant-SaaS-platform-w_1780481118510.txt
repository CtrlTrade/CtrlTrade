Build CtrlTrade® as a multi-tenant SaaS platform with two connected but separate tenant business applications:

1. CtrlTrade CRM
2. CtrlTradePos®

Important:
CtrlTradePos® is NOT part of the CRM.
CtrlTradePos® is a separate EPOS trade counter system connected to the CRM through shared customers, products, stock, invoices, payments and reports.

CTRLTRADEPOS® LICENSING
==================================================

Price:
£59.99 / till / month

Each till requires:
- licence key
- terminal ID
- tenant ID
- branch ID
- till name
- activation status
- last licence check timestamp

Licence states:
- Active
- Trial
- Suspended
- Expired
- Revoked
- Read Only

Licence validation flow:

POS app opens
↓
Checks licence key
↓
Validates tenant, branch and terminal
↓
If active, allow till login
↓
If expired or suspended, lock till or switch to read-only mode

Super Admin can:
- issue licence key
- revoke licence key
- suspend till
- reactivate till
- move licence to another branch
- view terminal activity
- view till billing

Tenant admin can:
- view till licences
- assign till to branch
- rename till
- request extra till
- deactivate till
