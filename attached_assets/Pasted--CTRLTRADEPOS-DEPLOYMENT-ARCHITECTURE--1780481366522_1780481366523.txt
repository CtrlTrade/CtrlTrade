==================================================
CTRLTRADEPOS® DEPLOYMENT ARCHITECTURE
==================================================

CtrlTradePos® is a modern EPOS platform that supports both:

1. Cloud Web POS
2. Installed Desktop POS

Both versions connect to the same CtrlTrade platform and database.

==================================================
OPTION 1
WEB POS
==================================================

Cloud-hosted browser-based EPOS.

Access URL:

https://app.ctrltrade.co.uk/pos

or

https://tenantname.ctrltrade.co.uk/pos

Supported browsers:

- Chrome
- Edge
- Safari
- Firefox

Supported devices:

- Windows PCs
- Windows Touchscreen POS
- Android POS
- Android Tablets
- iPads
- Surface Pro
- Mac Desktop
- Linux Desktop

Benefits:

- No installation
- Instant updates
- Works anywhere
- Multi-branch access
- Easy deployment
