Build CtrlTrade CRM Mapping System.

CtrlTrade is one SaaS CRM platform, but every tenant must see a version mapped to their industry, business type, shop/POS needs, workforce, branches, warehouse, compliance and add-ons.

SIGNUP QUESTIONS

1. Business details
- Company name
- Contact name
- Email
- Phone
- Address
- Website
- Company number optional
- VAT number optional

2. Industry / trade category
Tenant must choose one:
- Roofing
- Plumbing
- Electrical
- Gas / Heating
- Building / Construction
- Cleaning
- Landscaping
- Windows & Doors
- Flooring
- Painting & Decorating
- Bathroom / Kitchen Installer
- Supplier / Merchant
- Warehouse / Distributor
- Field Service Company
- Other

3. Business structure
Ask:
“What best describes your business?”

Options:
- Sole Trader
- Trade Business
- Trade Counter / Merchant
- Supplier / Distributor
- Field Service Company
- Multi-Branch Company
- Franchise Group

4. Trade shop / POS question
Ask:
“Do you operate a trade counter, showroom, warehouse, or physical shop?”

Options:
- Yes
- No

If YES:
- Enable CtrlTrade POS
- Ask number of tills
- Ask number of warehouse locations
- Ask if stock management is required

If NO:
- Hide all POS, till, retail, barcode and trade-counter features

5. Mobile workforce
Ask:
“Do you have field engineers or mobile staff?”

If YES:
- Enable field seats
- Mobile jobs
- Engineer diary
- Job dispatch
- Photo uploads
- Signatures
- Site checklists
- Route planning
- Offline job mode

If NO:
- Hide field-service modules

6. Appointments
Ask:
“Do customers book appointments with you?”

If YES:
- Enable booking calendar
- Customer booking portal
- Time slots
- Automated reminders
- Google Calendar sync
- Outlook sync

7. Branches
Ask:
“Do you operate multiple locations?”

If YES:
- Enable branch selector
- Branch dashboards
- Branch stock
- Inter-branch stock transfers
- Location-based permissions
- Branch reporting

8. VAT and accounting
Ask:
- VAT registered?
- Accounting software used?
- Invoice numbering format?
- Financial year start?

Enable:
- Xero
- QuickBooks
- Sage
- Stripe
- Manual accounting

9. Communication
Ask:
“How do you communicate with customers?”

Options:
- Phone
- SMS
- Email
- WhatsApp
- VoIP
- Customer portal

Enable selected modules:
- SMS notifications
- Email automation
- VoIP calls
- Call recording
- WhatsApp messaging
- AI call summaries

10. AI setup
Ask:
“What would you like AI to help with?”

Options:
- Quote writing
- Job summaries
- Email drafting
- Call summaries
- Scheduling optimisation
- Lead scoring
- Customer replies
- Invoice chasing
- Profitability insights

Enable selected CtrlAI modules.

SECURITY RULES

Critical:
- Every record must have tenant_id
- Tenant users can only access their tenant data
- No tenant can see another tenant’s CRM
- No tenant can see another tenant’s customers
- No tenant can see another tenant’s quotes
- No tenant can see another tenant’s invoices
- No tenant can see another tenant’s files
- API queries must always filter by tenant_id
- File storage must be tenant-scoped
- Role-based access required
- Audit logs required
- GDPR export required
- GDPR delete request required
- MFA support required
- Session/device logging required

FINAL GOAL

CtrlTrade should not feel like a generic CRM.

It should automatically become:
- Roofing CRM for roofers
- Plumbing CRM for plumbers
- Electrical CRM for electricians
- Cleaning CRM for cleaners
- Construction CRM for builders
- POS + inventory system for trade shops
- Warehouse system for suppliers
- Field service system for mobile engineers
- Multi-branch system for larger companies

One platform. Many mapped CRM experiences. Fully tenant-isolated.
